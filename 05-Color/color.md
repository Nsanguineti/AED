# Color
---
## Tipo de dato Color

* Color = {c = (r,g,b)/r,g,b ∈ ℕ ∧ r,g,b ∈ [0,255]}
---
## Conjunto de Operaciones

* F. Mezclar
  mezclar: color x color -> color
    Sean c1 = (r1,g1,b1) ∧ c2 = (r2,g2,b2)
    mezclar (c1,c2) = c3 = (r3,g3,b3) = ((r1+r2)/2,(g1+g2)/2,(b1+b2)/2)
